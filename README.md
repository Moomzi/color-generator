# 🎨 智能色彩搭配系统

## 项目介绍
这是一个为设计学学生打造的AI配色工具，基于DeepSeek API和Flask开发。

## 功能特点
- 🤖 AI智能配色生成
- 🎨 专业色彩搭配建议
- 📱 响应式设计界面
- 📋 一键复制颜色代码

## 使用方法
1. 输入设计主题（如"科技金融"、"春日花园"）
2. 点击"生成配色方案"
3. 获取5套专业配色方案
4. 点击色块复制HEX颜色代码

## 技术栈
- 后端：Python + Flask
- 前端：HTML + CSS + JavaScript
- AI：DeepSeek API
